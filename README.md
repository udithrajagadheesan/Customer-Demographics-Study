Customer Demographics Study

📊 Project Overview

Customer Demographics Study is an interactive Power BI dashboard designed to analyze customer demographic data and identify key trends across different customer segments. The project provides insights into customer characteristics such as age, gender, income, and purchasing behavior.

🎯 Objectives

- Analyze customer demographics
- Identify customer segments
- Visualize demographic trends
- Support data-driven decision making

🛠️ Tools & Technologies

- Power BI
- Microsoft Excel
- Data Visualization

📈 Dashboard Features

- Customer Age Analysis
- Gender Distribution
- Income-Based Segmentation
- Regional Insights
- Interactive Filters and Visualizations
